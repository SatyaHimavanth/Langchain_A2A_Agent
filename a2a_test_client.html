⚙️ Configure Push Webhook
Register a webhook URL to receive real-time notifications when tasks complete, fail, or change state.
💡 Tip: Context ID represents your conversation. All messages in this session will send notifications to the same webhook.
💡 Tip: Run a local webhook receiver or use ngrok to expose localhost
🚀 Super Simple Setup
Step 1: Set your webhook URL above (or keep http://localhost:10000/push)
Step 2: Click "Activate Webhooks"
Step 3: Send messages in Chat tab
Done! All messages in this conversation will send notifications

The Context ID stays the same for the whole conversation, so you only need to set this once!

For Real Webhooks:
Create a simple Python webhook receiver and run:
from fastapi import FastAPI
app = FastAPI()
@app.post("/push")
async def push(request: Request):
  data = await request.json()
  print(f"✓ {data['contextId']}: {data['status']['state']}")
  return {"ok": True}

# uvicorn app:app --port 10000
